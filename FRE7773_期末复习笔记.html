Lecture 1 — 数学基础回顾

1.1 矩阵代数 (Linear Algebra)

关键恒等式

  • 转置：\((\mathbf{AB})^\top = \mathbf{B}^\top\mathbf{A}^\top\)
  • 逆：\((\mathbf{AB})^{-1} = \mathbf{B}^{-1}\mathbf{A}^{-1}\)，\((\mathbf{A}^\top)^{-1} = (\mathbf{A}^{-1})^\top\)
  • Woodbury 恒等式（重要，常用于“小修正大矩阵”求逆）： \[(\mathbf{A}+\mathbf{BD}^{-1}\mathbf{C})^{-1} = \mathbf{A}^{-1} - \mathbf{A}^{-1}\mathbf{B}(\mathbf{D}+\mathbf{C}\mathbf{A}^{-1}\mathbf{B})^{-1}\mathbf{C}\mathbf{A}^{-1}\]
  • \((\mathbf{I}+\mathbf{AB})^{-1}\mathbf{A} = \mathbf{A}(\mathbf{I}+\mathbf{BA})^{-1}\)

Trace 与 Determinant

  • \(\operatorname{Tr}(\mathbf{A}) = \sum_i A_{ii}\)，循环性 \(\operatorname{Tr}(\mathbf{ABC}) = \operatorname{Tr}(\mathbf{BCA}) = \operatorname{Tr}(\mathbf{CAB})\)。
  • \(|\mathbf{AB}| = |\mathbf{A}||\mathbf{B}|\)，\(|\mathbf{A}^{-1}| = 1/|\mathbf{A}|\)。
  • 重要恒等式：\(|\mathbf{I}_N + \mathbf{ab}^\top| = 1 + \mathbf{a}^\top\mathbf{b}\)。

矩阵求导（必须背）

表达式 对 \(\mathbf{x}\) 的梯度
\(\mathbf{x}^\top\mathbf{a}\) 或 \(\mathbf{a}^\top\mathbf{x}\) \(\mathbf{a}\)
\(\mathbf{x}^\top \mathbf{A}\mathbf{x}\) \((\mathbf{A}+\mathbf{A}^\top)\mathbf{x}\)
若 \(\mathbf{A}\) 对称: \(\mathbf{x}^\top\mathbf{A}\mathbf{x}\) \(2\mathbf{A}\mathbf{x}\)
\(\|\mathbf{y}-\mathbf{X}\boldsymbol\theta\|_2^2\) \(-2\mathbf{X}^\top(\mathbf{y}-\mathbf{X}\boldsymbol\theta)\)

特征分解 / Eigendecomposition

  • 对称矩阵 \(\mathbf{A}\in\mathbb{R}^{M\times M}\) 可对角化为 \(\mathbf{A}=\mathbf{U}\boldsymbol\Lambda\mathbf{U}^\top\)，其中 \(\mathbf{U}\) 为正交矩阵。
  • \(\det(\mathbf{A}) = \prod_i \lambda_i\)；\(\operatorname{Tr}(\mathbf{A}) = \sum_i \lambda_i\)。

奇异值分解 / Singular Value Decomposition (SVD)

对任意 \(\mathbf{X}\in\mathbb{R}^{M\times N}\)： \[\mathbf{X} = \mathbf{U}\boldsymbol\Sigma\mathbf{V}^\top = \sum_{i=1}^{r} \sigma_i\,\mathbf{u}_i\mathbf{v}_i^\top\]

  • \(\sigma_1\ge\sigma_2\ge\dots\ge\sigma_p\ge 0\)；\(\mathbf{u}_i\) 为左奇异向量，\(\mathbf{v}_i\) 为右奇异向量。
  • 关键关系：\(\mathbf{X}^\top\mathbf{X} = \mathbf{V}\boldsymbol\Sigma^2\mathbf{V}^\top\)，\(\mathbf{X}\mathbf{X}^\top = \mathbf{U}\boldsymbol\Sigma^2\mathbf{U}^\top\)。
  • 截断 SVD（取前 \(k\) 项）= rank-\(k\) 最佳逼近（Eckart–Young）。

对称半正定 (PSD)

  • \(\mathbf{A}\succeq 0 \iff \mathbf{x}^\top\mathbf{A}\mathbf{x}\ge 0\;\forall\mathbf{x} \iff\) 所有 \(\lambda_i\ge 0\)。
  • Multivariate Gaussian 要求协方差矩阵 正定（特征值严格 > 0），否则密度未定义。

1.2 概率与统计

乘法法则与全概率

  • Product rule: \(p(X,Y) = p(Y\mid X)p(X)\)。
  • Total probability: \(p(X) = \sum_i p(X\mid Y_i)p(Y_i)\)（或积分形式）。
  • Bayes 定理：\(p(Y\mid X) = \dfrac{p(X\mid Y)p(Y)}{p(X)}\)。

一元高斯

\[\mathcal N(x\mid\mu,\sigma^2) = \frac{1}{\sqrt{2\pi\sigma^2}}\exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right),\quad \mathbb E[x]=\mu,\ \operatorname{Var}(x)=\sigma^2\]

多元高斯 (MVG)

\[\mathcal N(\mathbf{x}\mid\boldsymbol\mu,\boldsymbol\Sigma) = \frac{1}{(2\pi)^{d/2}|\boldsymbol\Sigma|^{1/2}}\exp\!\left(-\tfrac12(\mathbf{x}-\boldsymbol\mu)^\top\boldsymbol\Sigma^{-1}(\mathbf{x}-\boldsymbol\mu)\right)\]

  • Mahalanobis 距离：\(\Delta(\mathbf{x}) = \sqrt{(\mathbf{x}-\boldsymbol\mu)^\top\boldsymbol\Sigma^{-1}(\mathbf{x}-\boldsymbol\mu)}\)（推广 z-score）。
  • 椭圆等高线由 \(\boldsymbol\Sigma\) 的特征向量决定主轴方向。

MVG 的边缘 / 条件分布（重要！）

设 \(\begin{pmatrix}\mathbf{x}_1\\ \mathbf{x}_2\end{pmatrix}\sim\mathcal N\!\left(\begin{pmatrix}\boldsymbol\mu_1\\\boldsymbol\mu_2\end{pmatrix},\begin{pmatrix}\boldsymbol\Sigma_{11}&\boldsymbol\Sigma_{12}\\\boldsymbol\Sigma_{12}^\top&\boldsymbol\Sigma_{22}\end{pmatrix}\right)\)：

  • 边缘：\(\mathbf{x}_1\sim\mathcal N(\boldsymbol\mu_1,\boldsymbol\Sigma_{11})\)。
  • 条件：\(\mathbf{x}_1\mid\mathbf{x}_2 \sim \mathcal N(\boldsymbol\mu_{1\mid 2},\boldsymbol\Sigma_{1\mid 2})\)，其中 \[\boldsymbol\mu_{1\mid 2} = \boldsymbol\mu_1 + \boldsymbol\Sigma_{12}\boldsymbol\Sigma_{22}^{-1}(\mathbf{x}_2-\boldsymbol\mu_2),\qquad \boldsymbol\Sigma_{1\mid 2} = \boldsymbol\Sigma_{11}-\boldsymbol\Sigma_{12}\boldsymbol\Sigma_{22}^{-1}\boldsymbol\Sigma_{12}^\top\]
  • 条件协方差 不依赖 观测值 \(\mathbf{x}_2\)。
  • 若 \(\boldsymbol\Sigma_{12}=\mathbf{0}\)（独立），则条件分布 = 边缘分布。

仿射变换与卷积

  • \(\mathbf{x}\sim\mathcal N(\boldsymbol\mu,\boldsymbol\Sigma)\Rightarrow \mathbf{Ax}+\mathbf{b}\sim\mathcal N(\mathbf{A}\boldsymbol\mu+\mathbf{b},\mathbf{A}\boldsymbol\Sigma\mathbf{A}^\top)\)。
  • 独立高斯之和：\(\mathbf{x}+\mathbf{y}\sim\mathcal N(\boldsymbol\mu+\mathbf{m},\boldsymbol\Sigma+\mathbf{S})\)。

卡方与 Student’s t

  • 若 \(x_i\overset{iid}{\sim}\mathcal N(0,1)\)，则 \(\sum x_i^2 \sim \chi^2_n\)。
  • 若 \(\mathbf{A}\) 对称且幂等，\(\mathbf{x}\sim\mathcal N(\mathbf{0},\mathbf{I})\)，则 \(\mathbf{x}^\top\mathbf{A}\mathbf{x}\sim \chi^2_{\operatorname{Tr}(\mathbf{A})}\)。
  • \(t_n = \dfrac{x}{\sqrt{y/n}}\)，\(x\sim\mathcal N(0,1),\ y\sim\chi_n^2\) 独立。

Lecture 2 — Maximum Likelihood Estimation (MLE)

2.1 概念

给定 i.i.d. 样本 \(\{x_i\}_{i=1}^n\) 和参数模型 \(p(x\mid\theta)\)：

  • Likelihood: \(\mathcal L(\theta) = \prod_{i=1}^n p(x_i\mid\theta)\)
  • Log-likelihood: \(\ell(\theta) = \sum_i \log p(x_i\mid\theta)\)
  • MLE: \(\hat\theta_{\text{MLE}} = \arg\max_\theta \mathcal L(\theta) = \arg\max_\theta \ell(\theta)\)

取 log 的好处：乘积变求和；数值稳定；常使目标函数变凹（concave）。

直观：选择 \(\theta\) 使观察到的数据“看起来最典型”。

2.2 经典模型的 MLE 闭式解

模型 MLE 闭式解
Bernoulli(\(\theta\)) \(\hat\theta = \bar x_n = \tfrac{1}{n}\sum x_i\)
Binomial(\(n,\theta\)) \(\hat\theta = x/n\)（样本成功比例）
\(\mathcal N(\mu,\sigma^2)\) \(\hat\mu = \bar x_n,\ \hat\sigma^2_{\text{MLE}} = \tfrac{1}{n}\sum(x_i-\bar x)^2\)（有偏！）
无偏方差 \(\hat\sigma^2_{\text{unb}} = \tfrac{1}{n-1}\sum(x_i-\bar x)^2\)
MVG \(\mathcal N(\boldsymbol\mu,\boldsymbol\Sigma)\) \(\hat{\boldsymbol\mu}=\tfrac1n\sum\mathbf x_i,\ \hat{\boldsymbol\Sigma}=\tfrac1n\sum(\mathbf x_i-\hat{\boldsymbol\mu})(\mathbf x_i-\hat{\boldsymbol\mu})^\top\)

2.3 关键代码（Bernoulli MLE 数值验证）

import numpy as np
xs = np.random.binomial(1, 0.7, size=200)
theta_hat = xs.mean()       # MLE = sample mean
loglik = lambda t: np.sum(xs*np.log(t) + (1-xs)*np.log(1-t))

考点提醒：MLE 不一定无偏；高斯方差的 MLE 比无偏估计 small by factor \((n-1)/n\)。

Lecture 3 — Linear Regression / OLS

3.1 回归任务

  • 风险（risk）：\(R(f) = \mathbb E[(Y-f(\mathbf X))^2]\)。
  • 最优解（贝叶斯回归函数）：\(r(\mathbf x) = \mathbb E[Y\mid\mathbf X=\mathbf x]\)。
  • 误差分解：\(R(f) = \underbrace{\int (f(\mathbf x)-r(\mathbf x))^2 p(\mathbf x)d\mathbf x}_{\text{逼近误差}} + \underbrace{\int\operatorname{Var}(Y\mid\mathbf X=\mathbf x)p(\mathbf x)d\mathbf x}_{\text{不可约误差}}\)。

3.2 经验风险最小化 (ERM)

\[\hat f \in \arg\min_{f\in\mathcal F}\frac{1}{n}\sum_{i=1}^n (y_i-f(\mathbf x_i))^2\]

模型族 \(\mathcal F\) 太复杂会过拟合；要控制复杂度。

3.3 一维 OLS

模型：\(y_i = \theta_0 + \theta_1 x_i + \varepsilon_i\)，\(\varepsilon_i\overset{iid}{\sim}\mathcal N(0,\sigma^2)\)。

MLE = 最小二乘：在高斯噪声假设下，\(\arg\max\ell \iff \arg\min\sum(y_i-\theta_0-\theta_1 x_i)^2\)。

闭式解： \[\hat\theta_1 = \frac{\sum(x_i-\bar x)(y_i-\bar y)}{\sum(x_i-\bar x)^2} = \operatorname{Corr}(x,y)\cdot\frac{\sqrt{\operatorname{Var}(y)}}{\sqrt{\operatorname{Var}(x)}},\qquad \hat\theta_0 = \bar y - \hat\theta_1\bar x\]

3.4 多元 OLS（向量化）

设 design matrix \(\mathbf X\in\mathbb R^{n\times(p+1)}\)（首列为 1）： \[\hat{\boldsymbol\theta} = \arg\min_{\boldsymbol\theta}\tfrac12\|\mathbf y - \mathbf X\boldsymbol\theta\|_2^2\]

梯度：\(\nabla f(\boldsymbol\theta) = \mathbf X^\top(\mathbf X\boldsymbol\theta-\mathbf y)\)。

Normal equations：\(\mathbf X^\top\mathbf X\boldsymbol\theta = \mathbf X^\top\mathbf y\)。

Full-rank 解：\(\hat{\boldsymbol\theta} = (\mathbf X^\top\mathbf X)^{-1}\mathbf X^\top\mathbf y\)。

实战中绝不显式计算 \((\mathbf X^\top\mathbf X)^{-1}\)（昂贵 + 数值不稳定），而用 QR / SVD / numpy.linalg.solve。

3.5 几何性质 — Hat matrix

\[\hat{\mathbf y} = \mathbf X\hat{\boldsymbol\theta} = \mathbf H\mathbf y,\qquad \mathbf H = \mathbf X(\mathbf X^\top\mathbf X)^{-1}\mathbf X^\top\]

  • \(\mathbf H\) 对称 (\(\mathbf H^\top=\mathbf H\)) 且幂等 (\(\mathbf H^2=\mathbf H\)) ⇒ 是到 \(\mathbf X\) 列空间的正交投影矩阵。
  • 残差 \(\mathbf y-\hat{\mathbf y}\) 与 \(\mathbf X\) 的列空间正交。

3.6 统计性质

Gaussian linear model: \(\mathbf y = \mathbf X\boldsymbol\theta + \boldsymbol\epsilon,\ \boldsymbol\epsilon\mid\mathbf X\sim\mathcal N(\mathbf 0,\sigma^2\mathbf I)\)。

  • \(\hat{\boldsymbol\theta}\mid\mathbf X \sim \mathcal N(\boldsymbol\theta,\sigma^2(\mathbf X^\top\mathbf X)^{-1})\)
  • 无偏：\(\mathbb E[\hat{\boldsymbol\theta}]=\boldsymbol\theta\)。
  • 是 \(t\)-检验的基础。

Gauss–Markov 定理：在所有线性无偏估计量中，OLS 方差最小（BLUE）。不需要高斯假设。

Lecture 4 — Ridge & Lasso（正则化）

4.1 病态问题动机（OLS via SVD）

\[\hat{\boldsymbol\theta}_{\text{OLS}} = \mathbf X^+\mathbf y = \sum_{i=1}^r \frac{1}{s_i}\mathbf v_i(\mathbf u_i^\top\mathbf y)\]

  • 小奇异值 \(s_i\) ⇒ 估计被放大 ⇒ 高方差、out-of-sample 极差。
  • 高度相关特征会产生小 \(s_i\) — 金融常见（factor 多重共线性）。

4.2 Ridge Regression (Tikhonov, \(\ell_2\) 罚)

惩罚形式： \[\hat{\boldsymbol\theta}^{\text{rdg}}_\lambda = \arg\min_{\boldsymbol\theta}\|\mathbf y-\mathbf X\boldsymbol\theta\|_2^2 + \lambda\|\boldsymbol\theta\|_2^2\]

约束形式（等价）：\(\arg\min\|\mathbf y-\mathbf X\boldsymbol\theta\|^2 \text{ s.t. } \|\boldsymbol\theta\|_2^2\le T\)。

闭式解： \[\boxed{\hat{\boldsymbol\theta}^{\text{rdg}}_\lambda = (\mathbf X^\top\mathbf X + \lambda\mathbf I_p)^{-1}\mathbf X^\top\mathbf y}\]

SVD 视角：\(\hat{\boldsymbol\theta}^{\text{rdg}}_\lambda = \sum_i \frac{s_i}{s_i^2+\lambda}\mathbf v_i(\mathbf u_i^\top\mathbf y)\)。每个奇异值方向被乘以 \(\frac{s_i^2}{s_i^2+\lambda}\)（频率相关衰减）。

Kernel trick：\(\mathbf X^\top(\mathbf X\mathbf X^\top+\lambda\mathbf I_n)^{-1}\mathbf y = (\mathbf X^\top\mathbf X+\lambda\mathbf I_p)^{-1}\mathbf X^\top\mathbf y\)。当 \(n<p\)（高维数据）用左式更便宜。

正交特例：\(\mathbf X^\top\mathbf X=\mathbf I_p \Rightarrow \hat{\boldsymbol\theta}^{\text{rdg}} = \tfrac{1}{1+\lambda}\hat{\boldsymbol\theta}^{\text{OLS}}\)（统一线性收缩）。

偏差/方差： - \(\operatorname{Bias}(\hat{\boldsymbol\theta}^{\text{rdg}}_\lambda) = -\lambda(\mathbf X^\top\mathbf X+\lambda\mathbf I)^{-1}\boldsymbol\theta\) - \(\operatorname{Var}(\hat{\boldsymbol\theta}^{\text{rdg}}_\lambda) = \sigma^2(\mathbf X^\top\mathbf X+\lambda\mathbf I)^{-2}\mathbf X^\top\mathbf X\) - 极限：\(\lambda\to 0^+\) → OLS；\(\lambda\to\infty\) → \(\mathbf 0\)。

4.3 Lasso (\(\ell_1\) 罚)

\[\hat{\boldsymbol\theta}^{\text{lasso}}_\lambda = \arg\min_{\boldsymbol\theta}\|\mathbf y-\mathbf X\boldsymbol\theta\|_2^2 + \lambda\|\boldsymbol\theta\|_1\]

  • \(\ell_1\) ball 有“角”，contour 与 ball 接触点常落在轴上 ⇒ 稀疏解（自动 feature selection）。
  • 没有闭式解（需 coordinate descent / proximal / LARS）。

正交特例（soft-thresholding）： \[\hat\theta^{\text{lasso}}_j = \operatorname{sign}(\hat\theta^{\text{OLS}}_j)\bigl(|\hat\theta^{\text{OLS}}_j|-\lambda\bigr)_+\]

4.5 数据预处理与 Intercept

  • 必须先标准化特征（同尺度），否则不同尺度的系数被惩罚不一致。
  • Intercept 不加入惩罚（中心化数据后自动消失，或在 sklearn 用 fit_intercept=True）。

4.6 Train / Test Split & Cross-Validation

协议（必背）：

  1. 随机切 train (e.g. 80%) / test (20%)。
  2. 在 train 上做 K-fold CV 选超参 \(\lambda\)。
  3. 用最优 \(\lambda\) 在 完整 train 上重新训练。
  4. 最后 在 test 上评估一次（不重复评估！）。

K-fold 流程： - 把 train 平均分 \(K\) 折；轮流取 1 折为 validation，其余为训练。 - 对每个候选 \(\lambda\) 在 \(K\) 次 fold 上的 validation error 取平均，选最小的 \(\hat\lambda_{\text{CV}}\)。 - 常用 \(K=5\) 或 \(K=10\)；\(K=n\) 时为 leave-one-out。

Test error 通常呈 U 型曲线（train error 单调下降）—— 防止 underfit / overfit。

Lecture 5 — Classification & Logistic Regression

5.1 任务设置

  • 目标 \(Y\in\{0,1\}\)；条件分布 \(Y\mid\mathbf X\sim\text{Bernoulli}(p(\mathbf X))\)。
  • 直接用 linear regression 的问题：预测可能 \(\notin[0,1]\)；无概率解释。

5.2 Logit 链接 & 模型

Sigmoid (logistic): \(\sigma(z) = \dfrac{1}{1+e^{-z}} = \dfrac{e^z}{1+e^z}\)。

Logit: \(g(t) = \log\!\dfrac{t}{1-t}\)（\(\sigma\) 的反函数）。

模型： \[\log\!\frac{P(Y=1\mid\mathbf X)}{P(Y=0\mid\mathbf X)} = \beta_0 + \boldsymbol\beta^\top\mathbf X\] 等价于 \(P(Y=1\mid\mathbf x) = \sigma(\mathbf x^\top\boldsymbol\beta)\)。

其他链接：probit (\(\Phi\)), cloglog (\(1-e^{-e^t}\))。

5.3 极大似然估计

Likelihood（标签 \(\in\{0,1\}\)）： \[\mathcal L_n(\boldsymbol\beta) = \prod_{i=1}^n p_{\boldsymbol\beta}(\mathbf x_i)^{y_i}(1-p_{\boldsymbol\beta}(\mathbf x_i))^{1-y_i}\]

Negative log-likelihood (NLL)： \[\ell_n(\boldsymbol\beta) = -\frac{1}{n}\sum_i \bigl[y_i\,\mathbf x_i^\top\boldsymbol\beta - \log(1+e^{\mathbf x_i^\top\boldsymbol\beta})\bigr]\]

用 \(y_i\in\{-1,+1\}\) 时更紧凑： \[\ell_n(\boldsymbol\beta) = \frac{1}{n}\sum_i \log\!\bigl(1+e^{-y_i\,\mathbf x_i^\top\boldsymbol\beta}\bigr)\]

梯度：\(\nabla\ell_n(\boldsymbol\beta) = -\dfrac{1}{n}\sum_i \mathbf x_i\,(y_i-\sigma(\mathbf x_i^\top\boldsymbol\beta))\)。

Hessian （重要：证明凸性）： \[\nabla^2\ell_n(\boldsymbol\beta) = \frac{1}{n}\sum_i \sigma(\mathbf x_i^\top\boldsymbol\beta)(1-\sigma(\mathbf x_i^\top\boldsymbol\beta))\,\mathbf x_i\mathbf x_i^\top \succeq \mathbf 0\]

⇒ NLL 是 凸函数；任何局部极小即全局。
⇒ NLL 是 \(L\)-smooth，\(L = \tfrac{1}{4n}\lambda_{\max}\bigl(\sum_i \mathbf x_i\mathbf x_i^\top\bigr)\)（因 \(\sigma(t)(1-\sigma(t))\le 1/4\)）。

5.4 系数解释 — Odds Ratio

\(\text{OR}_j = e^{\beta_j}\)：\(X^{(j)}\) 增加 1 单位时 odds 的乘性变化。

\(\beta_j\) OR 含义
\(>0\) \(>1\) 提高 odds
\(=0\) \(=1\) 无影响
\(<0\) \(<1\) 降低 odds

陷阱：odds ratio ≠ probability ratio（仅在概率很小时近似相等）。

5.5 分类指标

Confusion matrix（二分类）：

Predicted 0 Predicted 1
Actual 0 TN FP
Actual 1 FN TP
指标 公式
Accuracy \(\dfrac{TP+TN}{TP+TN+FP+FN}\)
Misclassification rate \(1-\)Accuracy
Sensitivity = Recall = TPR \(\dfrac{TP}{TP+FN}\)
Specificity \(\dfrac{TN}{TN+FP}\)
FPR \(1-\)Specificity \(= \dfrac{FP}{FP+TN}\)
Precision \(\dfrac{TP}{TP+FP}\)
F1 \(2\cdot\dfrac{P\cdot R}{P+R}\)

ROC & AUC： - 改变阈值 \(\tau\)（决策规则 \(\hat y=\mathbf 1\{p\ge\tau\}\)），画 (FPR, TPR) 曲线。 - AUC = ROC 下面积；越接近 1 越好；\(0.5\) = 随机。

5.6 高维 / 正则化逻辑回归

\[\arg\min_{\boldsymbol\beta} \frac{1}{n}\sum_i \log(1+e^{-y_i\mathbf x_i^\top\boldsymbol\beta}) + \text{pen}(\boldsymbol\beta)\]

  • 罚项：\(\lambda\|\boldsymbol\beta\|_2^2\)（Ridge），\(\lambda\|\boldsymbol\beta\|_1\)（Lasso），Elastic Net。
  • 在 sklearn 中 LogisticRegression(C=1/lambda, penalty='l2'|'l1'|'elasticnet')。

5.7 Generalized Linear Models (GLM)

三件套： 1. Random component: \(Y_i\sim\) exponential family，\(\mathbb E[Y_i]=\mu_i\)。 2. Linear predictor: \(\eta_i = \mathbf x_i^\top\boldsymbol\beta\)。 3. Link function: \(g(\mu_i) = \eta_i\)。

模型 分布 Link
线性回归 Gaussian identity
逻辑回归 Bernoulli logit
Poisson 回归 Poisson log

Lecture 6 — Optimization for ML

6.1 为何需要优化算法

Naive grid search 在 \(d\) 维 \([0,1]^d\) 上需 \((1/\varepsilon)^d\) 次评估，\(d=100,\varepsilon=0.1\) 已经 \(10^{100}\) — 不可行。
解析解仅对 OLS 等少数模型存在，逻辑回归等需迭代算法。

6.2 凸性与平滑性

  • \(f\) 凸 ⇔ \(\nabla^2 f\succeq 0\)（如果二次可导）。凸函数局部最小 = 全局最小。
  • \(f\) 是 \(L\)-smooth ⇔ 梯度 \(L\)-Lipschitz ⇔ \(\lambda_{\max}(\nabla^2 f)\le L\)。

6.3 Descent Lemma（梯度下降的核心）

若 \(f\) 是 \(L\)-smooth： \[f(\boldsymbol\beta') \le f(\boldsymbol\beta) + \langle\nabla f(\boldsymbol\beta),\boldsymbol\beta'-\boldsymbol\beta\rangle + \frac{L}{2}\|\boldsymbol\beta'-\boldsymbol\beta\|^2\]

最小化右边的二次模型 ⇒ 步长 \(\eta = 1/L\)，得到 GD 更新。

6.4 (Batch) Gradient Descent

\[\boxed{\boldsymbol\beta^{(k+1)} = \boldsymbol\beta^{(k)} - \eta\,\nabla f(\boldsymbol\beta^{(k)})}\]

  • 每步成本 \(O(nd)\)（一个完整数据 pass）。
  • 对凸 + \(L\)-smooth 函数收敛速率 \(O(1/k)\)；强凸时 \(O(\rho^k)\)（线性）。

6.5 Stochastic Gradient Descent (SGD)

把目标写为 \(f(\boldsymbol\beta) = \frac{1}{n}\sum_{i=1}^n f_i(\boldsymbol\beta)\)。

\[\boldsymbol\beta^{(k+1)} = \boldsymbol\beta^{(k)} - \eta_k\,\nabla f_{i_k}(\boldsymbol\beta^{(k)}),\quad i_k\overset{\text{unif}}{\sim}\{1,\dots,n\}\]

  • 每步成本 \(O(d)\)（仅一个样本）。
  • 步长一般递减（如 \(\eta_k = \eta/\sqrt{k}\) 或 schedule \(\eta(t) = t_0/(t+t_1)\)）保证收敛。
  • 收敛是噪声 trajectory，不沿严格下降方向。

6.6 Mini-batch GD

每步用大小为 \(B\) 的 mini-batch（典型 32 / 64 / 128 / 256）： \[\nabla\hat f = \frac{1}{B}\sum_{i\in\text{batch}}\nabla f_i(\boldsymbol\beta)\]

  • 折中：方差比 SGD 小，比 BGD 便宜。
  • 利用 vectorization & GPU 并行，是深度学习的默认。

6.7 三种 GD 路径对比（Linear regression）

# Batch GD
for n in range(n_epochs):
    grads = X_b.T @ (X_b @ theta - y) / m
    theta = theta - eta * grads

# SGD
for epoch in range(n_epochs):
    for i in range(m):
        idx = rng.integers(m)
        xi, yi = X_b[idx:idx+1], y[idx:idx+1]
        grads = xi.T @ (xi @ theta - yi)
        eta = t0/(epoch*m + i + t1)            # learning schedule
        theta = theta - eta * grads

# Mini-batch GD
for epoch in range(n_epochs):
    perm = rng.permutation(m)
    for k in range(0, m, B):
        idx = perm[k:k+B]
        grads = X_b[idx].T @ (X_b[idx] @ theta - y[idx]) / B
        theta = theta - eta * grads

6.8 逻辑回归的 GD（套用上面 NLL 的梯度）

\[\nabla\ell_n(\boldsymbol\beta) = \frac{1}{n}\sum_i \bigl(\sigma(\mathbf x_i^\top\boldsymbol\beta) - y_i\bigr)\mathbf x_i\]

for k in range(n_iter):
    z = X @ beta                  # (n,)
    p = 1.0 / (1.0 + np.exp(-z))  # sigmoid
    grad = X.T @ (p - y) / n
    beta -= eta * grad

Lecture 7 — PCA

7.1 目的

Principal Component Analysis 寻找正交方向使投影后方差最大化： - 可视化（高维 → 2D / 3D） - 去噪（投影到 top-\(k\) 子空间再重构） - 压缩 / 加速下游模型 - 特征提取（金融中的“风险因子”）

7.2 数学公式

设中心化数据 \(\mathbf Y = \mathbf X - \mathbf 1\mathbf g^\top\)，\(\mathbf g\) 为列均值。协方差矩阵： \[\boldsymbol\Sigma = \tfrac{1}{n}\mathbf Y^\top\mathbf Y\]

第一主轴： \[\mathbf v_1 = \arg\max_{\|\mathbf v\|=1}\sum_i (\mathbf y_i^\top\mathbf v)^2 = \arg\max_{\|\mathbf v\|=1} \mathbf v^\top \boldsymbol\Sigma \mathbf v\]

用 Lagrange 法 ⇒ \(\boldsymbol\Sigma\mathbf v_1 = \lambda_1\mathbf v_1\)（\(\boldsymbol\Sigma\) 的最大特征向量）。

所有主成分：\(\boldsymbol\Sigma\mathbf v_i = \lambda_i\mathbf v_i,\ \lambda_1\ge\lambda_2\ge\dots\ge 0\)。
\(\mathbf v_k\) 在所有与 \(\{\mathbf v_1,\dots,\mathbf v_{k-1}\}\) 正交的方向中方差最大。
Variance explained by \(\mathbf v_k\) = \(\lambda_k\)。

7.3 投影与重构

  • 主成分得分（scores）：\(\mathbf c_j = \mathbf Y\mathbf v_j\)，矩阵形式 \(\mathbf C = \mathbf Y\mathbf V\)。
  • 取前 \(k\) 个：\(\mathbf C_k = \mathbf Y\mathbf V_k\)，\(\mathbf V_k = [\mathbf v_1,\dots,\mathbf v_k]\)。
  • 重构：\(\hat{\mathbf Y} = \mathbf C_k\mathbf V_k^\top = \mathbf Y\mathbf V_k\mathbf V_k^\top\)（rank-\(k\) 最佳逼近）。

7.4 标准化 PCA

变量尺度不一致时（金融中常见）： \[z_i^j = \frac{x_i^j - \bar x^j}{s^j},\qquad s^j = \sqrt{\sigma_j^2}\]

对 \(\mathbf Z\) 做 PCA，其协方差 $= Z^Z = $ 相关矩阵。
所有变量单位方差，PCA 不被高方差特征主导。

7.5 PCA 与 SVD 的关系（关键）

\(\mathbf Y = \mathbf U\boldsymbol\Sigma\mathbf V^\top\) ⇒ - \(\mathbf V\) 的列 = 主轴（特征向量 of \(\mathbf Y^\top\mathbf Y\)） - \(\lambda_j = s_j^2 / n\)（\(\boldsymbol\Sigma_{\text{cov}}\) 的特征值） - 主成分得分 = \(\mathbf Y\mathbf V = \mathbf U\boldsymbol\Sigma\)

为什么走 SVD：避免显式构造 \(\mathbf Y^\top\mathbf Y\)（贵 + 数值不稳）；对高维只需算 truncated SVD（前 \(k\) 个奇异向量）。sklearn 的 PCA 就是基于 SVD。

7.6 选择 \(k\)

  • Scree plot（肘部 / elbow 法）：寻找特征值衰减的拐点。
  • Kaiser 准则（标准化 PCA 中）：保留 \(\lambda_k > 1\) 的成分。
  • 累积解释方差比： \[I_k = \frac{\sum_{j=1}^k \lambda_j}{\sum_{j=1}^p \lambda_j}\;\;(\text{e.g. } I_k\ge 0.9)\]
  • 任务驱动：用 CV 在下游模型上选 \(k\)。

7.7 表现质量

  • 个体的表现：$^2(z_i,v_k) = $ 个体在第 \(k\) 主轴上的投影占其总方差的比例。
  • 变量的表现：\(r(\mathbf c_k,\mathbf z_j) = \sqrt{\lambda_k}\,v_{j,k}\)。
  • 标准化 PCA 中 \(\sum_{j=1}^p \lambda_j = p\)。

7.8 金融应用：Yield Curve PCA

对一组 swap rates / yield 收益率（每行一天，每列一个 maturity）做 PCA： 1. PC1 ≈ Level（曲线整体平移），通常解释 70-80%。 2. PC2 ≈ Slope（短端 vs 长端反向）。 3. PC3 ≈ Curvature（中部 vs 两端）。

import numpy as np
# data: shape (T_days, n_maturities)

Lecture 8 — K-Means & Gaussian Mixture Models

8.1 K-Means：目标函数

数据点 \(\mathbf x_i\in\mathbb R^K\)；目标找质心 \(\mathbf c_1,\dots,\mathbf c_Q\) 最小化 量化误差： \[J(\mathbf c_1,\dots,\mathbf c_Q) = \sum_{i=1}^I \min_{q=1,\dots,Q}\|\mathbf x_i-\mathbf c_q\|_2^2\]

全局最优 NP-hard（\(Q^I\) 种分配）。Lloyd 算法只能局部最优。

8.2 Lloyd 算法

  1. 初始化 \(\{\mathbf c_1,\dots,\mathbf c_Q\}\)。
  2. Assignment：\(\mathcal C_q = \{i: q = \arg\min_{q'} \|\mathbf x_i-\mathbf c_{q'}\|^2\}\)。
  3. Update：\(\mathbf c_q = \dfrac{1}{|\mathcal C_q|}\sum_{i\in\mathcal C_q}\mathbf x_i\)。
  4. 重复 2–3 直到 \(\mathcal C_q\) 不变。

每步严格不增加 \(J\) ⇒ 单调收敛到局部极小。

复杂度：\(O(Q\cdot n\cdot I\cdot s)\)（\(I\) 迭代数，\(s\) 距离计算成本）。

8.3 初始化方法

  • 随机选 \(Q\) 个点：可能差。
  • Farthest-point：先随机一个，再每次选离已选点最远的。对 outlier 敏感。
  • K-means++（推荐）：
    1. 均匀随机选 \(\mathbf c_1\)。
    2. 后续每次按 \(\Pr(\mathbf x_i)\propto \min_{q'<q}\|\mathbf x_i-\mathbf c_{q'}\|^2\) 加权采样。
    3. 期望意义上 \(\mathbb E[J]\le 8(\log Q+2)\,J^*\)（Arthur–Vassilvitskii 2006）。

8.4 选 \(Q\)

  • Elbow method：画 inertia \(J\) vs \(Q\) 曲线，看拐点。
  • Silhouette score（\(\in[-1,1]\)，越大越好）。
  • Gap statistic / Davies-Bouldin 等。

8.5 Gaussian Mixture Models (GMM)

K 个高斯成分的线性组合： \[p(\mathbf x) = \sum_{k=1}^K \pi_k\,\mathcal N(\mathbf x\mid\boldsymbol\mu_k,\boldsymbol\Sigma_k),\quad \sum_k \pi_k=1,\ \pi_k\ge 0\]

潜变量表示（1-of-K）：\(\mathbf z\in\{0,1\}^K,\ \sum z_k=1\)；\(p(z_k=1)=\pi_k\)；\(p(\mathbf x\mid z_k=1)=\mathcal N(\boldsymbol\mu_k,\boldsymbol\Sigma_k)\)。

Responsibility（关键）： \[\gamma(z_{nk}) = p(z_k=1\mid\mathbf x_n) = \frac{\pi_k\mathcal N(\mathbf x_n\mid\boldsymbol\mu_k,\boldsymbol\Sigma_k)}{\sum_j \pi_j\mathcal N(\mathbf x_n\mid\boldsymbol\mu_j,\boldsymbol\Sigma_j)}\]

8.6 EM 算法 for GMM

直接最大化 log-likelihood \(\sum_n\log\sum_k\pi_k\mathcal N(\dots)\) 没有闭式解（log 内有 sum）。EM 提供迭代方案。

Algorithm：

  1. 初始化 \(\boldsymbol\mu_k,\boldsymbol\Sigma_k,\pi_k\)。
  2. E-step：用当前参数计算 \(\gamma(z_{nk})\)。
  3. M-step（更新参数）： \[\boxed{\begin{aligned} N_k &= \sum_{n=1}^N \gamma(z_{nk})\\ \boldsymbol\mu_k^{\text{new}} &= \frac{1}{N_k}\sum_n \gamma(z_{nk})\mathbf x_n\\ \boldsymbol\Sigma_k^{\text{new}} &= \frac{1}{N_k}\sum_n \gamma(z_{nk})(\mathbf x_n-\boldsymbol\mu_k^{\text{new}})(\mathbf x_n-\boldsymbol\mu_k^{\text{new}})^\top\\ \pi_k^{\text{new}} &= \frac{N_k}{N} \end{aligned}}\]
  4. 计算 log-likelihood，未收敛回 2。

性质：单调不减（log-lik 不会下降），但只保证局部最优；GMM log-likelihood 非凸。

实践小贴士： - 用 K-means 结果初始化 EM 加速收敛（簇均值 → \(\boldsymbol\mu_k\)，簇 cov → \(\boldsymbol\Sigma_k\)，簇比例 → \(\pi_k\)）。 - 协方差可能退化（奇异），常加 regularization \(\boldsymbol\Sigma_k + \varepsilon\mathbf I\)。

8.7 K-Means vs GMM

K-Means GMM
簇形状 球形 椭圆（任意协方差）
分配 硬（hard） 软（responsibilities）
优化 Lloyd EM
复杂度 低 较高（每次计算 cov）
概率模型 否 是（可计算密度）
对偏离球形敏感 是 否

K-Means 可视为 GMM 的极限：\(\boldsymbol\Sigma_k = \sigma^2\mathbf I,\ \sigma\to 0\)。

9.1 神经网络基本概念

单个 neuron： \[y = f\!\left(\sum_i w_i x_i + b\right) = f(\mathbf w^\top\mathbf x + b)\] 其中 \(f\) 是 activation function。

多层网络（MLP / 全连接）： \[\mathbf h = f_1(\mathbf x\mathbf W_1 + \mathbf b_1),\quad \mathbf y = f_2(\mathbf h\mathbf W_2 + \mathbf b_2)\]

Hyperparameters：层数、每层 unit 数、激活函数、学习率、batch size、epochs。

9.2 Activation Functions

名称 公式 特点
Sigmoid \(\sigma(z) = 1/(1+e^{-z})\) 输出 \((0,1)\)；梯度饱和；二分类输出层
Tanh \(\tanh(z)\) 输出 \((-1,1)\)；零中心；仍饱和
ReLU \(\max(0,z)\) 计算便宜；隐藏层默认；可能 dead neurons
Softmax \(\sigma_i = e^{z_i}/\sum_j e^{z_j}\) 多分类输出层；总和为 1
LogSoftmax \(\log\sigma_i\) 数值更稳；与 NLLLoss 配套

重要：必须有非线性激活，否则多层网络等价单层线性映射。

任务 Loss PyTorch

B.1 概念辨析（高频）

  1. MLE vs OLS：在高斯噪声假设下，OLS = MLE。一般情况下两者不一定等价。
  2. MLE 高斯方差是有偏的 (\(\hat\sigma^2_{\text{MLE}}=\frac{1}{n}\sum(\cdot)\))，无偏估计要用 \(\frac{1}{n-1}\)。
  3. OLS 不需要高斯假设就是 BLUE（Gauss–Markov）；置信区间/t-检验 才需要 Gaussian。
  4. Ridge 不会把系数变 0；Lasso 会；Ridge 有闭式解，Lasso 没有。
  5. CV 应只在 train 上做；test 只用 1 次。
  6. L1 / L2 惩罚都需要预先标准化特征；intercept 不应被惩罚。
  7. Logistic regression 的 NLL 是凸函数（Hessian PSD），所以局部 = 全局最优。
  8. OR ≠ probability ratio（仅小概率近似）。
  9. PCA 假设方差代表“信息”；要先中心化 / 标准化。
  10. PCA 不是监督方法，不考虑标签；可能丢掉对预测有用但低方差的方向。
  11. K-Means 假设球形等大簇；尺度敏感（要标准化）；对初始化敏感（用 k-means++）。
  12. EM for GMM 单调收敛到局部最优；可用 K-Means 初始化 EM。
  13. K-Means 是 GMM 的极限（\(\boldsymbol\Sigma_k = \sigma^2\mathbf I\)，\(\sigma\to 0\)，硬分配）。
  14. CrossEntropyLoss 不要再 softmax；NLLLoss 必须配 log_softmax。
  15. 每个 batch 必须 optimizer.zero_grad()，否则梯度累加。

B.2 公式必背清单

  • 一元 OLS：\(\hat\theta_1 = \dfrac{\sum(x_i-\bar x)(y_i-\bar y)}{\sum(x_i-\bar x)^2},\ \hat\theta_0 = \bar y - \hat\theta_1\bar x\)。
  • 多元 OLS：\(\hat{\boldsymbol\theta} = (\mathbf X^\top\mathbf X)^{-1}\mathbf X^\top\mathbf y\)。
  • Ridge：\(\hat{\boldsymbol\theta}_\lambda = (\mathbf X^\top\mathbf X + \lambda\mathbf I)^{-1}\mathbf X^\top\mathbf y\)。
  • Lasso 正交解：\(\hat\theta_j = \operatorname{sign}(\hat\theta^{\text{OLS}}_j)(|\hat\theta^{\text{OLS}}_j|-\lambda)_+\)。
  • Hat 矩阵：\(\mathbf H = \mathbf X(\mathbf X^\top\mathbf X)^{-1}\mathbf X^\top\)（对称 + 幂等）。
  • Sigmoid 导数：\(\sigma'(z) = \sigma(z)(1-\sigma(z))\)。
  • Logistic NLL 梯度：\(\nabla\ell = \tfrac{1}{n}\sum_i (\sigma(\mathbf x_i^\top\boldsymbol\beta)-y_i)\mathbf x_i\)。
  • GD 更新：\(\boldsymbol\beta\gets\boldsymbol\beta - \eta\nabla f(\boldsymbol\beta)\)。
  • PCA：\(\boldsymbol\Sigma\mathbf v_k=\lambda_k\mathbf v_k\)；通过 SVD \(\mathbf Y=\mathbf U\boldsymbol\Sigma\mathbf V^\top\) 得到 \(\mathbf V\)。
  • K-Means 目标：\(\sum_i \min_q\|\mathbf x_i-\mathbf c_q\|^2\)。
  • GMM responsibility：\(\gamma(z_{nk})=\dfrac{\pi_k\mathcal N(\mathbf x_n\mid\boldsymbol\mu_k,\boldsymbol\Sigma_k)}{\sum_j \pi_j\mathcal N(\mathbf x_n\mid\boldsymbol\mu_j,\boldsymbol\Sigma_j)}\)。
  • GMM M-step：\(N_k=\sum_n\gamma_{nk}\)；\(\boldsymbol\mu_k=\tfrac1{N_k}\sum_n\gamma_{nk}\mathbf x_n\)；\(\pi_k=N_k/N\)。

B.4 速答模板

  • “什么是 overfitting？” → 训练误差很低但测试误差很高；模型过于复杂以至于学到了噪声；通过 regularization、CV、增加数据、early stopping 等缓解。
  • “为什么要交叉验证？” → 单次 holdout 估计有高方差；CV 在多个折上平均，估计更稳定，并能用于模型选择。
  • “Ridge 与 Lasso 怎么选？” → 想要稀疏 / 自动特征选择 → Lasso；想要稳定预测、特征高度相关 → Ridge；都想要 → Elastic Net。
  • “PCA 的几何意义？” → 在最大化方差的正交方向上做投影；等价于在 column space 上找 rank-\(k\) 最佳逼近（Frobenius 范数）。
  • “K-Means 与 GMM 区别？” → K-Means 硬分配 + 球形簇；GMM 概率模型 + 软分配 + 任意椭圆形簇。
  • “为什么神经网络要非线性激活？” → 否则任意层数等价于一层线性映射，无法逼近非线性函数。
